# Smart Mall Cinematic Reels

This project explores the Smart Mall concept through cinematic navigation and
interaction videos. It turns still architectural visualizations into vertical
reels that show how visitors discover information, make decisions, and move
through a future mall.

Learn more about the concept on the
[Smart Mall website](https://kirill-lebedev-lab.github.io/smart-mall/).

The visual narrative focuses on distributed navigation, interactive screens,
architectural movement, and camera-driven storytelling. Each scene communicates
both the spatial design and the intended visitor experience.

The videos are produced with a reusable shot-based generation framework.
See [scripts/README.md](scripts/README.md) for its architecture, rendering
pipeline, and build commands.
